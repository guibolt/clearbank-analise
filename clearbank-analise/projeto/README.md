# Projeto Análise Financeira - ClearBank

Este notebook faz a leitura da base de transações, trata os dados e monta um resumo mensal da análise. A ideia aqui é ter um fluxo simples, direto e fácil de acompanhar.

## Como executar

Abra `analysis.ipynb` no Colab ou no Jupyter e rode todas as células em ordem. Se preferir executar localmente, basta abrir a pasta `projeto` no VS Code e seguir a mesma sequência.

## O que o notebook gera

Quando a execução termina, o notebook cria `relatorio.json` com o resumo mensal e com as transações que chamam mais atenção na análise.

## Arquivos do projeto

- `analysis.ipynb`: notebook principal da análise.
- `transacoes.csv`: base usada nos testes.
- `relatorio.json`: saída gerada ao final da execução.
